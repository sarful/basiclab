# Learner Dashboard Planning

This folder contains placeholder planning scaffolds for the learner dashboard
navigation sections.

Current live learner dashboard work still renders from:

- `src/auth/DashboardView.tsx`

These files are intentionally minimal and do not change routing yet. They exist
to prepare future implementation for:

- My Courses
- Lessons
- Assignments
- Exams
- Progress
- Certificates
- Calendar
- Messages
- Settings

Suggested future route pattern:

- `/dashboard/my-courses`
- `/dashboard/lessons`
- `/dashboard/assignments`
- `/dashboard/exams`
- `/dashboard/progress`
- `/dashboard/certificates`
- `/dashboard/calendar`
- `/dashboard/messages`
- `/dashboard/settings`
